A custom quote cart system for event services.

== Description ==

Event Quote Cart provides a way for sales team members to create quotes for multiple event services. It integrates with HivePress and WooCommerce to provide a seamless quoting experience.

Features:
* Add multiple items to quote cart
* Custom quote forms
* Role-based access control
* Multi-language support
* Integration with existing booking systems

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/event-quote-cart` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings through the 'Quote Cart' menu in WordPress.

== Frequently Asked Questions ==

= Who can use the quote cart? =

By default, only administrators and users with the 'ventas-team' role can use the quote cart functionality.

= Can I customize the quote form? =

Yes, the quote form can be customized through template overrides and filters.
